Developer Docs
REST API Reference

Integrate BounceIQ predictions directly into your Python backend, CI pipeline, or analytics stack.

Authentication
All requests must include your API key in the Authorization header.
# Python example import requests headers = { "Authorization": "Bearer YOUR_API_KEY", "Content-Type": "application/json" }
POST /v1/predict/bounce Predict bounce rate for a page ▼
import requests res = requests.post( "https://api.bounceiq.io/v1/predict/bounce", headers=headers, json={ "url": "/pricing", "page_type": "pricing", "load_time_ms": 2850, "cta_count": 1, "scroll_depth": 28, "session_duration_sec": 42 } ) data = res.json() # → { "bounce_probability": 0.81, "risk_level": "HIGH", ... }
POST /v1/batch/predict Predict up to 100 pages at once ▼
import requests res = requests.post( "https://api.bounceiq.io/v1/batch/predict", headers=headers, json={ "pages": [ { "url": "/pricing", "load_time_ms": 2850 }, { "url": "/checkout", "load_time_ms": 1200 } ] } )
GET /v1/model/info Get model version and feature list ▼
import requests res = requests.get( "https://api.bounceiq.io/v1/model/info", headers=headers ) # → { "model_version": "v2.4.0", "accuracy": "94.2%", ... }
GET /v1/analytics/summary Fetch analytics summary for a date range ▼
import requests res = requests.get( "https://api.bounceiq.io/v1/analytics/summary", headers=headers, params={ "start": "2025-03-01", "end": "2025-03-31", "site_id": "site_abc123" } )
DELETE /v1/sites/{site_id}/data Delete all tracking data for a site ▼
import requests res = requests.delete( "https://api.bounceiq.io/v1/sites/site_abc123/data", headers=headers ) # Returns 204 No Content on success
